# Monstermaze PaperMC Java plugin
### Minecraft 1.12.11

This plugin is a minigame being developed by tinesone.
It's an asymmetrical PVP minigame where 1 player takes control of a monster _(like the warden)_ and has to destroy the other players, who in turn must find a way to defeat the monster.

It features random maze generation.


## Planned features

1. Random maze generation ✔️
2. fully featured lobby
3. Fully build tilesets
4. Different kinds of monsters to play as
5. Fun loot
6. In world shop
7. Player classes
8. Custom tilesets support
